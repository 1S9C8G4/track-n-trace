# Track'n Trace

Track and trace for Pilship booking numbers.

## Getting Started

Make sure you have redis running and use Node.js v7.5.0.

* git clone https://github.com/1S9C8G4/track-n-trace.git
* cd track-n-trace
* npm install
* npm run build
* npm start
* open http://localhost:3000/
